Re-Design Studierendenwerk Siegen Website

Renting through Studierendenwerk Website is very tiring and complicated procedure and requires lot of paper work, exchange of pdfs between stakeholders through email.

Figma prototyping | Experience Design | Project coordination

The Problem

Renting through Studierendenwerk Website is very tiring and complicated procedure and requires lot of paper work, exchange of pdfs between stakeholders through email. Lot of uncertainity on availability of rooms which create problem for the expats or even the natives travelling from far distance.

The Goal

Improve the experience of new comer or already existing users if Studierendenwerk website to book or change the room using design thinking.

Team Size : 5
Project type : University course project
Duration : 3 weeks
Supervised by : Mr. Hussain Abid Syed
Tools Used :
