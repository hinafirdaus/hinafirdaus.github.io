Nexus

bringing long distance couples together...

The Product

Nexus helps in connecting couples living in long distance with each other by playing nostalgia games in their relationship.

The Problem

Couples living afar from each other. Craving intimacy and playful way to connect with each other. Whenever you have a feeling of loneliness and want to play a game to go back in memory.

The Goal

Couples in long distance can play games at their home and outside.

My Roles

Sketch, Wireframe, Rapid prototyping, User flow, team coordination, Presentation

Team Size : 10
Project type : University course project
Duration : 4 weeks
Supervised by : Prof. Dr. Marc Hassenzahl
Tools Used :

Design Process

Wireframe Sketch

I crafted the interface magic while my teammate sculpted the device. From logos to screen size, every detail's on paper.

Rapid Prototyping

When couple turn on the Nexus device it shows three icon. There are three types of game function couples can play. Find physical gift or item that couples shared with each other, second game is make memory and play together, and third part is add game so your partner can play.

While playing find object game together. One of the couple can program the game and other couple in different time zone can see notification and open the screen. Take photo and if the photo matched it shows green tick with text and audio message from your partner to tell you that you performed well.

Couple can also program the game. Add object game where couple can add clues based on the memory they have created earlier. In this game can be played in different time. Another game is to create memory together and they can play in real time.

Reflections

  • Only a single type game can be played, which may lead to loss of interest.
  • The lack of a full keyboard/touch screen may make it hard for user operations.
  • The hardware resembles to an inferior/low-grade product.
  • Coupled camera and recorder may make it costly for its single application.
  • It is reproducible as a mobile application.
  • The gift/objects can be lost or unavailable at that time.
  • There is limit for the objects that can be available with a person, which restricts the game.
